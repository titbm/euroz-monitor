# EUROZ Monitor Bot

Telegram бот для мониторинга статуса контрактов EUROZ и cEUROZ на Sepolia.

## Установка

```bash
cd euroz-monitor
npm install
```

## Настройка

1. Создай бота у @BotFather в Telegram и получи токен
2. Скопируй `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```
3. Заполни `.env`:
   - `TELEGRAM_BOT_TOKEN` — токен от BotFather
   - `TELEGRAM_CHAT_ID` — ID чата для автоматических уведомлений (опционально)

## Запуск

```bash
npm start
```

## Команды бота

- `/start` — приветствие и список команд
- `/status` — текущий статус контрактов
- `/subscribe` — получить свой Chat ID для настройки алертов

## Как работает

- Бот проверяет функцию `paused()` на обоих контрактах каждые 5 минут
- Если статус изменился — отправляет уведомление в указанный чат
- Любой пользователь может проверить статус командой `/status`
